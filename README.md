# Marke Lungau

## API

**Install**
```
$ yarn install
```

**Seed mongoDB**
```
$ npm run setup
```

**Start**
```
$ npm start     //local dev environment
$ npm run prod  //production environment
```

**API Testing**
```
$ npm test
```

**Doc**

Documentation is found at [api.berneau.at](http://api.berneau.at).
```
$ npm run doc
```
